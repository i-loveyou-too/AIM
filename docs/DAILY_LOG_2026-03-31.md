## 내일 바로 실행할 명령어 (복붙용)
```bash
# 1) 서버 접속
ssh clover@192.168.99.99

# 2) 프로젝트 이동 + venv
cd ~/projects/AIM
source .venv/bin/activate

# 3) env 로딩
set -a
source .env.local
set +a

# 4) DB 직접 접속 테스트 (비밀번호는 실제값)
PGPASSWORD='실제비밀번호' psql \
  -h "$DJANGO_DB_HOST" \
  -p "$DJANGO_DB_PORT" \
  -U "$DJANGO_DB_USER" \
  -d "$DJANGO_DB_NAME" \
  -c "select 1;"

# 5) 성공하면 .env.local 반영 후 Django 연결 테스트
python backend/manage.py shell -c "from django.db import connection; c=connection.cursor(); c.execute('select 1'); print(c.fetchone())"

# 6) 백엔드 실행
./scripts/start-backend.sh

# 7) (새 터미널) 프론트 실행
cd ~/projects/AIM
./scripts/start-frontend.sh
```

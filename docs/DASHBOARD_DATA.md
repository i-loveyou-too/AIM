# Aim ON Dashboard Data 문서

이 문서는 `src/lib/mock-data/dashboard.ts` 안에 있는 대시보드 데이터를 항목별로 정리한 문서입니다.

## 문서 경계
- 이 문서: 대시보드 화면 데이터 항목/필드 기준
- [MOCK_DATA.md](./MOCK_DATA.md): mock 제거 현황/실데이터 전환 상태

## 요약

대시보드는 다음 8개 묶음으로 구성됩니다.

| 묶음 | 역할 |
| --- | --- |
| `summaryStats` | 상단 요약 카드 4개 |
| `dashboardHero` | 인사말 영역 |
| `aiInsights.classInsights` | 반별 인사이트 목록 |
| `aiInsights.studentInsight` | 학생 인사이트 목록 |
| `examAlert` | 시험 알림 기본 카드 |
| `examSchools` | 학교 선택형 시험 데이터 |
| `todaySchedule` | 오늘 수업 일정 |
| `recentActivity` | 최근 활동 |
| `quickActions` | 빠른 실행 버튼 |
| `examHighlights` | 시험 임박 학생 하이라이트 |

## 사용 규칙

- 카드 수와 문구를 바꾸면 이 문서도 같이 바꾼다.
- 숫자는 화면에서 보이는 값과 문서 값이 같아야 한다.
- 학교/반별 선택형 데이터는 대시보드에서 먼저 확인하고 문서에 반영한다.
- 새로운 요약 카드나 일정 카드가 생기면 이 문서에 먼저 추가한다.
